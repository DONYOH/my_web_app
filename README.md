---

## 🌟 Aperçu du Projet

Ce site vitrine a été conçu pour allier **féerie** et **crédibilité institutionnelle**. L'objectif est de transformer des visiteurs curieux en acquéreurs de créatures légendaires grâce à une expérience utilisateur (UX) fluide et un design haut de gamme.

### ✨ Points Forts :
* **Hero Section Immersive :** Un message fort avec un effet de texte dégradé.
* **Catalogue Dynamique :** Gestion visuelle des stocks (Vente vs Location).
* **Mobile-First :** Entièrement responsive pour les clients consultant le catalogue depuis leurs écuries.
* **Performance :** Chargement ultra-rapide via l'utilisation de Tailwind CSS en JIT (Just-in-Time).

## 📖 Structure du Code

Le projet est contenu dans un fichier unique `index.html` pour une maintenance simplifiée :

1.  **`<nav>`** : Barre de navigation flottante avec effet de flou (*backdrop-blur*).
2.  **`#accueil`** : Section d'appel à l'action avec typographie imposante.
3.  **`#catalogue`** : Grille de produits avec effets de zoom au survol.
4.  **`#services`** : Liste des garanties (Assurance Magique, Livraison Pégase).
5.  **`#contact`** : Formulaire de capture de leads qualifiés.

---

## 🚀 Déploiement Rapide

Pour visualiser le site immédiatement :
1. Télécharge le fichier `index.html`.
2. Ouvre-le dans n'importe quel navigateur moderne (Chrome, Firefox, Safari).
3. **Aucune installation de dépendances n'est requise** (utilisation de CDN).

---

## 🛡️ Maintenance & Entretien

Pour modifier les couleurs principales, changez simplement les classes utilitaires de Tailwind dans le code :
* Primaire : `purple-600`
* Accent : `amber-400`
* Fond : `slate-900` (Mode sombre)
